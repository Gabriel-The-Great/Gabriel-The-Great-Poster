# Security Policy

## Supported Versions

We currently support the latest version of **Ngong Mawien** with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in **Ngong Mawien**, please do **not** open a public issue. 

Instead, please send an email to **[drngongny@gmail.com](mailto:drngongny@gmail.com)** with the subject line "Security Report".

We take security seriously and will respond to your report within 48 hours. Once verified, we will work on a fix and credit you (if you wish) in the release notes.

Thank you for helping keep **Ngong Mawien** safe!
